# LayerOS

Light liquid-glass demo for the LayerOS concept.

LayerOS is presented here as a web-only prototype: a calm OS-like canvas where a user can describe the workspace they want, open a starter template, and watch LayerOS generate a role-specific workflow surface.

## Run

```bash
npm install
npm run dev
```

Then open the local URL shown by Vite.

## Demo Flow

- Start on the blank LayerOS canvas.
- Use the `+` menu to choose a starter workflow.
- Try `Client Call` to show the live transcript, suggested response, and next steps.
- Type commands like `run a client call`, `research a company`, `make a finance report`, or `plan my day`.
